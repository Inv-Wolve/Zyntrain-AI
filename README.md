# Zyntrain AI

**Zyntrain AI** learns your habits, plans your day, and syncs everything with Google Calendar — so your tasks stay on track, even when you’re not.

---

## Overview
Zyntrain AI is an experimental **AI-powered task manager** built to automate planning and time management.  
It integrates directly with **Google Calendar** to intelligently organize your schedule, prioritize your tasks, and adapt to your work habits over time.

This project is currently in **Beta**, and under active development.

---

## Features
-  **AI Scheduling:** Automatically arranges and prioritizes tasks based on context and workload.
-  **Google Calendar Sync:** Keeps your events, reminders, and tasks in perfect alignment.
-  **Smart Insights:** Learns your habits and suggests focus sessions or breaks dynamically.
-  **Upcoming Integrations:** Slack and Notion support (coming soon).

---

## Tech Stack
- **Node.js** (Backend)
- **Express.js** for API endpoints
- **Google Calendar API** for sync & event management
- **OpenAI / LLM integration (planned)** for adaptive task management

---

## Current Status
Zyntrain AI is currently in **Alpha/Beta** phase — focused on testing core functionality, improving the scheduling logic, and refining the user experience.

This project is **not yet open for public deployment**, but is shared for demonstration and learning purposes.
